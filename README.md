# Финальное задание — ML basics (AAA)

Краткое описание
- Проект: задача предсказания физических параметров товаров (вес, высота, длина, ширина) по метаданным и опционально картинкам.

Структура репозитория
- [final-task-aaa.ipynb](final-task-aaa.ipynb) — полный ноутбук с EDA, обучением моделей и экспериментами.
- [README.md](README.md) — этот файл.

Коротко об экспериментах и результатах
- EDA и лог-преобразование таргетов для стабильности обучения.
- Базовая модель: CatBoost на `microcat_name` + `title` (встроенная обработка текста)
- Улучшения: FastText + PCA для title + CatBoost.
- Попытки добавить CLIP-эмбеддинги для изображений — ухудшили качество

Примечание
- Ноутбук содержит комментарии и замечания, а также идеи и подробный вывод
